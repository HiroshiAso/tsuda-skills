# 勤務表登録手順

このスキルは、勤務表システムにログインし、当日の勤怠申請を行うための手順を定義します。

## 手順

1.  **ログイン**
    *   ブラウザで [https://kintai.ahc-net.co.jp/CP01/login](https://kintai.ahc-net.co.jp/CP01/login) にアクセスする。
    *   以下の認証情報を入力する:
        *   **法人ID**: `000308`
        *   **ID**: `tsuda_hi`
        *   **パスワード**: `Hiroshi4428`
    *   ログインボタンをクリックする。

2.  **申請画面への移動**
    *   「入力・申請」メニューをクリックする。
    *   「勤務表入力」をクリックする。

3.  **申請の作成**
    *   カレンダーまたはリストから「当日の日付」を探し、その行にある「入力」ボタンをクリックする。
    *   入力欄が表示されたら、開始時間が`09:00`、終了時間が`18:00`（または適切な時間）になっていることを確認する。

4.  **申請の保存**
    *   入力行の右側にある「保存」ボタンをクリックする。
    *   処理が完了するまで数秒待機する。

5.  **完了確認**
    *   画面上部に「◯/◯の勤務表を登録しました」というメッセージが表示されたこと、または日付行に勤務時間が反映されたことを確認して完了とする。

> [!WARNING]
> このスキルにはハードコードされたパスワードが含まれています。実際の運用環境では環境変数など安全な方法で管理することを推奨します。
