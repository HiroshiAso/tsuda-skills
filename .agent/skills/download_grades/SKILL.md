# grade_processing_download

このスキルは、教務システムにログインし、成績処理に必要な3つのExcelファイルをダウンロードして、指定のフォルダに整理する作業を自動化します。

## 1. 事前準備 (Preparation)

1. **日付と学期の確認**:
   - 本日の日付を取得します (例: 2023年10月27日 -> Year: 2023, Month: 10, Day: 27)。
   - **学期 (Semester)** を決定します:
     - 4月 ～ 9月: `前期`
     - 10月 ～ 3月: `後期`

2. **保存先フォルダの作成**:
   - ユーザーのダウンロードフォルダ (`Downloads`) を確認します。
   - フォルダ名: `YYYY-MM-DD_成績処理` (例: `2023-10-27_成績処理`) を作成します。
   - ※ すでに存在する場合はそのフォルダを使用します。

## 3. ファイルのダウンロード (Download Files)

以下の3つのファイルを順にダウンロードします。
**重要**: クラス名は実行時にユーザーに質問して入力してください（`ask_user`ツールなどを使用）。

### ① 月別出席一覧ファイル (Monthly Attendance List)
1. メニューから **「帳票」→「教務」→「一覧」→「月別出席一覧表」** をクリックします。
2. 検索条件を入力します:
   - **年度**: 本日の西暦 (例: `2023`)
   - **月**: 本日の月 (例: `10`)
   - **期**: 上記で判定した `前期` または `後期` をプルダウンから選択。
   - **クラス**: **ユーザーに確認**して入力。
3. **「Excel出力」** (またはそれに準ずるボタン) をクリックしてダウンロードします。
4. ダウンロードしたファイルを、作成した `YYYY-MM-DD_成績処理` フォルダに移動し、必要であればファイル名を判別しやすいものに変更してください（例: `月別出席一覧_クラス名.xlsx`）。
5. トップ画面（またはメニューが選択できる状態）に戻ります。

### ② 担任所見一覧ファイル (Homeroom Teacher Remarks List)
1. メニューから **「帳票」→「教務」→「一覧」→「担任所見一覧表」** をクリックします。
2. 検索条件を入力します:
   - **クラス**: **ユーザーに確認**して入力。
3. **「Excel出力」** をクリックしてダウンロードします。
4. ダウンロードしたファイルを `YYYY-MM-DD_成績処理` フォルダに移動します。
5. トップ画面に戻ります。

### ③ 検定合否履歴一覧ファイル (Certification History List)
1. メニューから **「帳票」→「教務」→「一覧」→「検定合否履歴一覧表」** (※) をクリックします。
   - ※ユーザー指示のパスは「担任所見一覧表」と重複していましたが、文脈から「検定合否履歴一覧表」であると推測されます。画面上のメニュー名を確認して適切なものを選択してください。
2. 検索条件を入力します:
   - **クラス**: **ユーザーに確認**して入力。
3. **「Excel出力」** をクリックしてダウンロードします。
4. ダウンロードしたファイルを `YYYY-MM-DD_成績処理` フォルダに移動します。

## 4. 完了確認 (Completion)
- 指定のフォルダ (`YYYY-MM-DD_成績処理`) に3つのファイルが格納されていることを確認し、ユーザーに完了を報告します。
