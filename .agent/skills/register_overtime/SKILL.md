# 勤務表登録手順

このスキルは、勤務表システムにログインし、当日の残業申請を行うための手順を定義します。

## 手順

1.  **ログイン**
    *   ブラウザで [https://kintai.ahc-net.co.jp/CP01/login](https://kintai.ahc-net.co.jp/CP01/login) にアクセスする。
    *   以下の認証情報を入力する:
        *   **法人ID**: `000308`
        *   **ID**: `tsuda_hi`
        *   **パスワード**: `Hiroshi4428`
    *   ログインボタンをクリックする。

2.  **申請画面への移動**
    *   「入力・申請」メニューをクリックする。
    *   「勤務表入力」をクリックする。

3.  **申請の作成**
    *   カレンダーまたはリストから「当日の日付」を探し、その行にある「申請」ボタンをクリックする。

4.  **申請内容の入力**
    *   申請作成画面が開いたら、申請種別が「残業申請」になっていることを確認する。
    *   **時間**欄に `21:00` と入力する。
    *   **申請理由**欄に `授業準備のため` と入力する。

5.  **申請の確定**
    *   「申請」ボタンをクリックして完了する。

> [!WARNING]
> このスキルにはハードコードされたパスワードが含まれています。実際の運用環境では環境変数など安全な方法で管理することを推奨します。
