# フォンブース予約手順 (Outlook)

ユーザーの要望に基づき、Outlookカレンダーを使用して1号館のフォンブースを予約します。

## 1. Outlookカレンダーを開く

1.  ブラウザで [https://outlook.office.com/calendar/](https://outlook.office.com/calendar/) を開きます。
    *   **注意**: 既にログインしているプロファイルを使用してください。ログインが必要な場合はユーザーにログインを促してください。
    *   もしMicrosoftのログイン情報を求められた場合は、以下の情報を使用してください：
        *   **Email**: `tsuda_hi@asojuku.ac.jp`
        *   **Password**: `e8xb4CzH`

## 2. 空き状況の確認

1.  **新しい予定の作成**:
    *   「新しいイベント」ボタンをクリック、またはカレンダー上で予約したい日時をクリックして予定作成画面を開きます。
    *   ユーザーが日付と時間を指定します（例：「明日の14時から」）。

2.  **場所の検索**:
    *   「場所の検索」または「会議室または場所を追加」欄に **「1号館」** と入力して検索します。
    *   検索結果から「フォンブース」に関連する会議室を探します（例: `1号館-1F-フォンブースA` など）。
    *   **重要**: ユーザーが指定した時間帯で「空き」となっているブースを確認します。既に予約が入っているブースは除外してください。

## 3. ユーザーへの提案と選択

1.  **リストの提示**:
    *   空いているフォンブースのリストをユーザーに提示します。
    *   例:
        ```text
        以下のフォンブースが空いています：
        1. 1号館-1F-フォンブースA
        2. 1号館-2F-フォンブースB
        どちらにしますか？
        ```
2.  **ユーザーの選択**:
    *   ユーザーからの回答（「Aでお願いします」「1番で」など）を待ちます。

## 4. 予約の確定

1.  **場所の追加**:
    *   ユーザーが選んだフォンブースを「場所」フィールドに確実に追加します。

2.  **内容の入力**:
    *   **タイトル**: ユーザーから指定があればそのタイトル（例：「Web会議」）、なければ「フォンブース利用」など適切なタイトルを入力します。
    *   **時間**: 指定された予約日・予約時間になっているか最終確認します。

3.  **保存**:
    *   「保存」または「送信」ボタンをクリックして予約を完了します。

4.  **完了報告**:
    *   予約が完了したことをユーザーに報告します。
    *   例: 「1号館-1F-フォンブースAで予約しました。」
