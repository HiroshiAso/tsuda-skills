# Googleカレンダー予定登録

ユーザーの指示に従い、`hiroshinkg@gmail.com` のGoogleカレンダーに予定を追加します。

## アカウント確認
1. https://calendar.google.com/ にアクセスする。
2. 現在ログインしているアカウントが `hiroshinkg@gmail.com` であることを確認する。
   - アイコンのツールチップやアカウントメニューで確認する。
   - 異なる場合はアカウントスイッチャーで切り替える。

## 予定の作成手順

1. **予定作成の開始**
   - 左上の「作成」ボタン(または＋ボタン)をクリックし、「予定」を選択する。
   - またはショートカットキー `c` を使用する。

2. **日時の設定**
   - ユーザーの指示（プロンプト）に基づき、以下のルールを適用する：
     - **開始時間が指定された場合**
       - 開始日時：ユーザーが指定した時間
       - 終了日時：開始時間の **1時間後** (1時間の枠を確保する)
     - **「終日」と指定された場合**
       - 「終日」のチェックボックスをオンにする。
     - **時限（1限、2限など）が指定された場合**
       - **1限**: 09:30 開始
       - **2限**: 11:15 開始
       - **3限**: 13:30 開始
       - **4限**: 15:15 開始

3. **その他の設定**
   - **タイトル**: ユーザーが指定した件名を入力する。件名が明示されていない場合は「予定」とするか、文脈から推測される内容を入力する。

4. **保存**
   - 「保存」ボタンをクリックして登録を完了する。
